# 要件定義書：MAJIレスシステム (MVP)

## 1. サービス概要

ユーザーの意見や意思決定案に対し、役割の異なる3体のサブエージェントが「マジレス（忖度のない客観的指摘）」を行い、意思決定の精度を高めるための深掘り支援アプリ。

## 2. システムアーキテクチャ

論理（ロジック）と表現（トーン）を分離した5エージェント構成。

### 2-1. エージェント構成

| レイヤー | エージェント名 | 役割           | 思考ロジック                                  |
| :------- | :------------- | :------------- | :-------------------------------------------- |
| 分析層   | MELCHIOR       | 肯定的視点     | メリット最大化、機会損失回避、促進的提案。    |
| 分析層   | BALTHASAR      | 批判的視点     | 論理的欠陥の指摘、リスク抽出、失敗シナリオ。  |
| 分析層   | CASPER         | 中立的視点     | 社会的通念、倫理性、メタ認知的な俯瞰。        |
| 統合層   | Agent B        | 論理統合・審議 | 3者の整合性確認、不足情報の特定、合議判定。   |
| 出力層   | Agent C        | トーン変換     | Agent Bの結論を保持し、指定のペルソナで出力。 |

## 3. 機能要件

### 3-1. 審議・合議ロジック

- 合意形成ルール（設定選択式）:
  - 多数決: 2票以上の賛成で「可決」判定。
  - 全会一致: 3票全ての賛成が必要。1票でも反対があれば「否決」判定。
- コンテクスト管理:
  - 「意思決定台帳（Decision Ledger）」方式。毎ターンの合意・未決事項を統合要約して次ターンへ継承。
  - 保持期間：最終更新から1週間（168時間）。

### 3-2. 深掘り（逆質問）アルゴリズム

- 目的: 手段（How）の前に、目的（Why）と成功の必須要素を問い直す。
- 挙動: 3者の意見が対立した「ピボットポイント」を特定し、判断に必要な情報をユーザーに逆質問する。

### 3-3. 例外処理（ちゃぶ台返し対応）

- 矛盾検知: 過去の確定事項と矛盾する入力を検知した場合、Agent Cが「自覚の確認」をユーザーに行う。
- リセットフロー: 前提変更を認める場合、台帳を初期化しRound 1から再開。過去の議論は微かな「バイアス」としてのみ残す。

## 4. UI/UX要件（モバイル最適化）

### 4-1. 視覚演出

- レイヤード・スタック: 3者の思考プロセスを垂直方向のリアルタイム・ストリームで表示。
- ステータスバー: 固定ヘッダーに3エージェントの同調状態（シンクロ率）を六角形アイコンで表示。
- 判定演出: 可決/否決時に全画面オーバーレイとハプティクス（振動）によるフィードバック。

### 4-2. 回答フロー（シンクロ・エディタ）

- コンテクスト・ドロワー: 入力中に3者の過去の発言をオーバーレイ参照・引用可能。
- ハイブリッド・入力: 自由記述に加え、数値やトグルによる構造化データを収集。

## 5. ペルソナ（出力トーン）定義

- Default: 機械的、箇条書き、感情を排したレポート形式。
- Strict (司令官): 威圧的、厳格、責任を問う口調。
- Sarcastic (皮肉屋): 知的な煽り、比喩を用いた欠陥の指摘。

## 6. 利用環境要件

本システムは、以下の環境においてシームレスに動作することを要件とする。

- PC環境: - 3カラム表示による情報の同時並行的な視認性を最大化する。
  - キーボードショートカットによる迅速なコンテキスト引用・操作をサポートする。
- スマートフォン（モバイル）環境:
  - 縦長画面に最適化した「レイヤード・スタック構造」を採用する。
  - ハプティクス（振動）やスワイプジェスチャーを駆使した直感的な操作感を実現する。
  - 外出先での隙間時間における「思考の壁打ち」を想定した軽量なUIレスポンスを維持する。

## 7. 客観的分析と予測（事実と予測）

### 【事実】

- マルチエージェントの有効性: LLMの役割を分散させることで、単一プロンプトよりも論理性と事実確認の精度が向上する（Chain-of-Debate手法）。
- 情報のボトルネック: 長大な履歴はLLMの精度を下げるため、統合ステート（台帳）方式は長期的な議論において計算効率と精度のバランスに優れる。

### 【予測】

- ユーザー心理: 「マジレス」という厳しいフィードバックに対し、ユーザーは「自分の案が真剣に吟味されている」という心理的充足を得やすく、リテンション向上に寄与する。
- モバイル体験: リアルタイム演出は魅力だが、通信環境や端末スペックによる遅延が課題となる可能性があるため、非同期処理の最適化が必要になる。
